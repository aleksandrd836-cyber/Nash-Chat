# 🎮 Vibe — Инструкция по запуску

Это пошаговая инструкция для запуска чата. Следуй строго по шагам — всё получится.

---

## 📋 Что тебе понадобится

- Компьютер с Windows
- Интернет
- ~20 минут времени

---

## 🟦 ШАГ 1 — Установить Node.js (один раз)

Node.js — это программа, без которой приложение не запустится.

1. Открой браузер и перейди на сайт: **https://nodejs.org**
2. Нажми большую зелёную кнопку **"Download Node.js (LTS)"**
3. Запусти скачанный файл и нажимай **"Next"** до конца установки
4. После установки **перезагрузи компьютер**

> ✅ Проверка: открой **PowerShell** (нажми Win+R, введи `powershell`, Enter)
> и напечатай: `node -v`
> Должно появиться что-то вроде `v20.x.x`

---

## 🟦 ШАГ 2 — Настроить базу данных Supabase (один раз)

Supabase — это бесплатная облачная база данных, где хранятся сообщения.

### 2.1 Создать аккаунт

1. Открой **https://supabase.com**
2. Нажми **"Start your project"** → войди через Google или создай аккаунт
3. Нажми **"New project"**
4. Заполни:
   - **Name**: `vibe` (или любое название)
   - **Database Password**: придумай пароль (запомни его!)
   - **Region**: выбери `West EU (Ireland)` — ближайший к России
5. Нажми **"Create new project"** и подожди ~2 минуты

### 2.2 Создать таблицы (структуру базы данных)

1. В левом меню нажми на иконку **"SQL Editor"** (значок `</>`)
2. Нажми кнопку **"New query"**
3. Открой папку с проектом на компьютере:
   `C:\Users\Александр\.gemini\antigravity\scratch\discord-clone\`
4. Найди файл **`supabase-schema.sql`** и открой его Блокнотом
5. Выдели весь текст (Ctrl+A), скопируй (Ctrl+C)
6. Вставь в поле SQL Editor в браузере (Ctrl+V)
7. Нажми зелёную кнопку **"Run"** (или F5)
8. Снизу должно появиться `Success. No rows returned`

### 2.3 Включить Realtime (для мгновенных сообщений)

1. В левом меню нажми **"Database"** → **"Replication"**
2. Найди строку **`messages`** и переключи тумблер в положение **ON**

### 2.4 Получить ключи доступа

1. В левом меню нажми на шестерёнку **"Project Settings"**
2. Выбери **"API"**
3. Скопируй и сохри в блокнот два значения:
   - **Project URL** — выглядит как `https://abcdefgh.supabase.co`
   - **anon public** (в разделе "Project API keys") — длинная строка `eyJhbGci...`

## 🟦 ШАГ 3 — Настроить файл с ключами

1. Открой папку проекта:
   `C:\Users\Александр\.gemini\antigravity\scratch\discord-clone\`
2. Найди файл **`.env.example`**
3. Скопируй этот файл и назови копию **`.env`** (без `.example`!)

   > ⚠️ Если Windows не даёт переименовать файл без расширения — открой PowerShell и выполни:
   > ```
   > cd "C:\Users\Александр\.gemini\antigravity\scratch\discord-clone"
   > copy .env.example .env
   > ```

4. Открой файл **`.env`** Блокнотом
5. Замени значения своими ключами из Шага 2.4:

```
VITE_SUPABASE_URL=https://ТВОЙ_URL.supabase.co
VITE_SUPABASE_ANON_KEY=ТВОЙ_ANON_KEY
```

6. Сохрани файл (Ctrl+S)

## 🟦 ШАГ 4 — Установить зависимости и запустить

1. Открой **PowerShell** (Win+R → `powershell` → Enter)
2. Перейди в папку проекта — скопируй и вставь эту команду:
   ```
   cd "C:\Users\Александр\.gemini\antigravity\scratch\discord-clone"
   ```
3. Установи зависимости (нужно только **один раз**):
   ```
   npm install
   ```
   Подожди — это займёт ~1-2 минуты
4. Запусти приложение:
   ```
   npm run dev
   ```
5. Появится текст с адресом, например:
   ```
   ➜  Local:   http://localhost:5173/
   ➜  Network: http://192.168.1.5:5173/
   ```

6. Открой браузер и перейди по адресу **http://localhost:5173**

🎉 **Приложение запущено!**

## 🟦 ШАГ 6 — Дать доступ друзьям (по локальной сети)

Если друзья находятся в **той же сети Wi-Fi** — они могут зайти по адресу из строки `Network:` (например `http://192.168.1.5:5173`).

### Если друзья далеко — нужен ngrok:

1. Зайди на **https://ngrok.com**, создай бесплатный аккаунт
2. Скачай ngrok, распакуй архив
3. Открой PowerShell в папке с ngrok и выполни:
   ```
   .\ngrok config add-authtoken ТВОЙ_ТОКЕН
   .\ngrok http 5173
   ```
4. Появится публичная ссылка вида `https://xxxx.ngrok-free.app`
5. Отправь эту ссылку друзьям — они смогут зайти с любого устройства

## ❓ Частые проблемы

| Проблема | Решение |
|----------|---------|
| `node` не найден | Перезагрузи компьютер после установки Node.js |
| Сообщения не приходят | Проверь что в `.env` правильно указаны ключи Supabase |
| Белый экран | Открой консоль браузера (F12) и посмотри на ошибки — напиши мне |
| Друг не может подключиться | Убедись что ngrok запущен и ссылка верная |
| Порт занят | Перезапусти компьютер или закрой другие программы |
